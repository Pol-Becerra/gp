# CRM Service

Servicio de gestión de relaciones con empresas y contactos.

## Funcionalidades

- CRUD de entidades (empresas)
- Gestión de múltiples direcciones, teléfonos y emails
- Gestión de contactos (personas) asociados a empresas
- Historial de interacciones

## Archivos

- `index.js` - Lógica central del CRM
- `entities.js` - Gestión de entidades
- `contacts.js` - Gestión de contactos
