# Task Management Service

Servicio de gestión de tickets y tareas automáticas.

## Funcionalidades

- Creación automática de tickets por validaciones fallidas
- Asignación inteligente de tareas a gestores
- Seguimiento de estados de tickets
- Priorización basada en SLA

## Archivos

- `index.js` - Motor de tareas
- `assignment.js` - Lógica de asignación inteligente
- `priority.js` - Algoritmo de priorización
